## CREATE SCHEMA
```SQL
CREATE SCHEMA 스키마명 AUTHORIZATION 소유권자;
```
## CREATE DOMAIN
```SQL
CREATE DOMAIN 도메인명 AS 데이터타입
DEFAULT 기본값
CONSTRAINT 제약조건명 CHECK (범위값);
```

```SQL
CREATE TABLE 테이블명
속성명 데이터타입 DEFAULT 기본값 NOT NULL
PRIMARY KEY
UNIQUE
FOREIGN KEY ~ REFERENCES

```

```SQL
CREATE (UNIQUE) INDEX 인덱스명
```

## ALTER TABLE
```SQL
ALTER TABLE 테이블명 ADD 속성명 데이터타입;
```
